2.维度问题

在使用concatenate() 、vstack()、hstack()时，需要保持维度一致。

 # (4,)  和  (4,1) 区别 是什么
 >>> t=np.linspace(1,3,num=4)
>>> t.shape
(4,)
>>> t[:,np.newaxis]
array([[1.        ],
       [1.66666667],
       [2.33333333],
       [3.        ]])
>>> t[:,np.newaxis].shape
(4, 1)

如上面的例子。

np.dot(x,y)
"""
如果x和y是一维的向量，那么计算内积。
如果有一个是二维的向量，那么按照2 矩阵乘法处理，此时需要满足，x必须是2维的，且x.shape[1]=y.shape[0]
"""

>>> np.dot(b,a)

"""
* 如果两者的维度一模一样，则，相当于一一对应的乘法；否则很复杂。

用 * 请慎重。
基本等同于 multiply
"""
>>> a*b   # b*a

inner : 不能处理复数的情况
